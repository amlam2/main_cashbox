# Формирование файла Excell по денежной потребности для главной кассы

**Название скрипта:** money_demand.

**Назначение скрипта:** формирование одного структурированного и удобного для восприятия файла Excell из нескольких автоматически сгенерированных файлов по денежной потребности для главной кассы.

Скрипт формирует структурированный, наглядный и удобный для восприятия файл Excell из нескольких автоматически сгенерированных "1С:Бухгалтерия" файлов по денежной потребности для главной кассы Берёзовского РУПС.


Скрипт написан на **Python 3.8**.


Используются модули, не входящие в стандартную поставку интерпретатора:
        - xlrd;
        - xlwt.
